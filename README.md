# flour
Flour Is A Simple Library Made for the Coders who want to have a sweet tooth while coding. Made For Python. Even C++ Or Lua if you know how to do that kind of stuff.


# How to use flour
For python users:
```
from flour import *
```

For Lua users:
```
require('Flour')
```
For C++ Users:
```
#include <CONT/Bread.h>
```
